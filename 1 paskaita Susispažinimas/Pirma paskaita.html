Tagų sąrašas

  • p - paragraph - Paragrafa, skirtas rašyti pastraipai
  • h1 -h6 - Heading1 - Heading6 - Antraštės, skirtos rašyti pavadinimams ir skiriasi teksto dydžiu (1-6)
  • img - image - Paveiksliukas, įkelti paveiksliukui, naudojamas su src (source) ir alt (jei neteisingas source, rodo alt)
  • a - hypertext anchor (link) - Nuoroda, nukelti į nurodytą puslapį
  • hr - Thematic Break (horizontal line), sukuria linija (eilutes lužį)
  • br - Line break - Eilutės lūžis
  • ul- unordered list - nerikiuotas sąrašas
  • ol - rikiuotas sąrašas
  • li - list item, rikiuotam arba nerikiuotam sąrašui
  • meta - skirtas nurodyti puslapio meta duomenis
  • title- skirtas nurodyti puslapio tabo pavadinimą

Atributų sąrašas

  • lang - language - nurodo kokia kalba vyrauja tame tag'e
  • charset - Character set - Simbolių rinkinys, nurodo kokią simbolių biblioteką naudoti
  • src - source - kelias nurodantis iki failo
  • alt - alternative text - Alternatyvus tekstas paveiksliuko neužkrovimo atveju
  • width - Nurodo elemento plotį
  • height - Nurodo elemento aukštį
  • href - hyper link reference - hyperteksto nuoroda - Nuoroda į puslapį arba failą
  • target - Nurodo kur atidaryti nuorodą

  • CTRL SHIFT C - puslapyje prie elementų rodo jų developer tool informacija
  • CTRL F - Atidaro paieškos sistemą
  • TAB - tabuliuoja pažymėtą kodą į dešinę
  • Shift TAB - tabuliuoja pažymėta kodą į kairę
