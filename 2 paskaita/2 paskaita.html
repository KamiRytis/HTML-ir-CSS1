Formos

  • form - Forma
  • input - įvestis
  • select - Pasirinkimas iš variantų
  • option - Pasirinkimo variantai
  • textarea - Didelio teksto kiekio įvestis
  • label - Etiketė prie įvesties/pasirinkimo
  • optgroup - Pasirinkimo variantų grupė
  • button - mygtukas
  • fieldset - Formos laukas
  • legend - Formos lauko pavadinimas

Atributai

  • type - Input tag'o tipas
  • for - Nurodo kuriam įvesties/parinkties laukui yra priskirta etiketė
  • id - unikalus pavadinimas, į kurį nurodo for atributas
  • name- įvesties lauko vardas, kuris bus perduotas back'ui kartu su tos įvesties reikšme
  • cols - columns - stulpelių kiekis textarea elemente
  • rows - Eilučių kiekis textarea elemente
  • placeholder - Tekstas esantis įvesties lauke, kuris pradingsta pradedant pildyti įvesties lauką
  • value - įvesties reikšmė, kuri perduodama į backend (serverį)
  • label - Priskiria Colgroup'ui pavadinimą
  • selected - numatytasis select tag'o viduje esantis option'as
  • minlength - Trumpiausia galima įvestis
  • maxlength - Ilgiausia galima įvestis
  • min - Mažiausia leistina įvestis
  • max - Didžiausia leistina įvestis
  • step - Žingsnis, kuriuo didėja skaičius
  • pattern - Reikalauja, kad įvesties laukelyje būtų parašytas nurodytas formatas (formatas nurodomas regex (regular expression)
  • checked - Numatytasis radio arba checkbox pasirinkimas
  • span - išskyria elementą arba jo dalį
