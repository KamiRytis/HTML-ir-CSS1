CSS rašymo būdai

Inline - linijinis

Linijinis stilius yra, kai elementui, kurį stilizuojame suteikiame style atributą ir jo viduje aprašome stilius. Neturi perpanaudojimo galimybės ir yra skirtas tiktais testavimui.

Internal - Vidinis

Vidinis stilius yra, kai HTML failo viduje, head tag'o dalyje pasirašome style tag'a ir tame style tag'e rašome CSS kodą. Turi šiek tiek perpanaudojimo galimybių ( nes turi selektorius) ir dažniausiai naudojamas testavimui.

Išorinis stilius yra rašomas atskirame CSS faile ir yra prijungiamas prie HTML failo naudojant link tag'ą. Turi visas reikalingas/norimas perpanaudojimo galimybbes, yra plačiausiai naudojamas ir galutinio projekto variante, turėtų likti tik šis stiliaus rašymo būdas.

Basic Selectors

Element / Tag

Selektinami visi tag'ai pagal nurodytą elementą.
Pvz.: h2 -> parinks visus h2 elementus esančius HTML'e.

Class

Klasė sukuriame kokiu tik nori logišku ir trumpu pavadinimu. CSS'e tam, kad nurodytume, jok kuriame klasę - sugalvoto pavadinimo pradžioje parašome tašką. HTML;e norint priskirti klasę elementui, tam elementui suteikiama atributą "class" ir jam priskiriame sugalvotą klasės pavadinimą be taško.
Pvz.: CSS'e: .klasesVardas; HTML'e: <class="klasesVardas">
Klasės turi daug su daug santykį. Vienas elementas gali turėti daugelį klasių ir viena klasė gali būti priskirta daugeliui elementų.

ID

ID sukuriame kokiu tik norime logišku ir trumpu pavadinimu. CSS'e tam, kad nurodytume, jok kuriame ID-sugalvoto pavadinimo pradžioje parašome groteles. HTML'e norint priskirti ID elementui, tam elementui suteikiame atributą "id" ir jam priskiriame sugalvotą ID pavadinimą be grotelių.
Pvz.: CSS'e #idPavadinimas; HTML'e: <pid="idPavadinimas">
ID turi vienas su vienu santykį. Vienas elementas gali turėti TIK vieną ID. Ir vienas ID turėtų būti priskirtas TIK vienam elementui.

Colors

Names

Spalvos gali turėti vardus. Vardinių spalvų yra ~140. Norint priskirti elementui spalvą, priskiri tos spalvos vardą. Pvz.: red; blue; green; light

RGB

RGB- Red Green Blue. Kiekvienos spalvos stiprumo (kiekio) galima idėti nuo 0 iki 255 ir šitaip išgauti bet kurią spalvą.
Pvz.: rgb(0,0,0)- juoda; rgb(255,0,0) - raudona;...
RGBA - Red Green Blue Alpha. Alpha nurodo spalvos matomumą ir jis gali būti nuo 0 iki 1 tūkstantąją paklaida.
Pvz.:rgba(0, 255, 0, 0) - Nematoma (green); rgba(0, 255,0,1)- pilnai matoma (green);...

Hex

Hex - hexadecimal - šešioliktainis.0-9 ir a-f Kiekvienos iš spalvos (RGB) galima nurodyti nuo 00 iki ff ir šitaip išgauti betkurią spalvą.
kaičiavimas šešioliktainėje sistemoje: 00 01 02 03 04 05 06 07 08 09 0a 0b 0c 0d 0e 0f 10 11 12 13 .....19 1a 1b ... 1f 20 21 22.. 9f a0 a1 a2 ... a9 aa ab ac ad ae af b0...ff
Pvz.:#000000 - juoda; #ff0000 - raudona; ...
Trumpiniai: #000 - juoda; #f00 raudona. Spalvos matomumas su Hex: gale prisideda dar du šešioliktainiai simboliai, kur 00 - permatomas, ff - pilnai matomas.
Pvz.: #0f00 (pilnas #00ff0000) - nematoma (žalia); #00ff00ff - pilnai matoma (žalia)...

Positions

Static

Statinė pozicija nieko nedaro. Nei pakeičia elemento poziciją, nei suteikia jam galimybę naudotis top, left, right, bottom, z-index stiliais.
Statinė pozicija gali būti suteikiama kaip placeholder'is arba panaudota tam, kad nuimti kokią kitą poziciją.

Relative

Relative pozicija pati jokių stilių nesuteikia. Suteikia galimybę naudotis top, left, right, bottom, z-index stiliais. Nurodo ribas kažkur jo viduje esančiam elementui su absolute pozicija.
Top, left, right arba bottom nustatymas pastumia elemmentą nuo jo pradinės pozicijos, per nurodytą atstumą, nuo nurodytos kraštinės.

Absolute

Elementas su absolute pozicija išeina iš puslapio tėkmės. Suteikia galimybę naudotis top, left, right, bottom, z-index stiliais.
Top, left, right arba bottom nustatymas pastumia elementą, per nurodytą atstumą, nuo nurodytos kraštinės, nuo pradinio viewport (ekrano vaizdo) ARBA nuo artimiausio protėvinio elemento, kuris turi relatyviąją poziciją.

Fixed

Elementas su fixed pozicija išeina iš puslapio tėkmės. Suteikia galimybę naudotis top, left, right, bottom, z-index stiliais.
Fiksuoja elementą matymo lauke (ekrane) ir jis juda kartu su ekranu kai vartotojas scroll'ina per puslapį.
Top, left, right arba bottom nustatymas nurodo, nuo kurios viewport(ekrano vaizdo) vietos jis atsitraukia per nurodytą atstumą, nuo nurodytos kraštinės.
